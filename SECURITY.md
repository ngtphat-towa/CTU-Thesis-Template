# Security Policy

## Supported Versions

The following versions of the template are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please open an issue or contact the maintainers directly. As this is a document template, security concerns usually involve the scripts or CI/CD workflows included.

We aim to respond to all reports within 48 hours.
